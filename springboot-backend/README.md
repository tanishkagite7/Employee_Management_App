# Employee Management App - Backend

This is the **Spring Boot backend** for the Employee Management Application. It provides REST APIs to manage employee data.

---

## Features

- CRUD operations for employees (Create, Read, Update, Delete)
- Connects to a MySQL database using Spring Data JPA
- Runs on **Java 17** with Spring Boot 3.x
- Maven project for dependency management

---

## Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

---

## Prerequisites

- Java 17+
- Maven 3.9+
- MySQL database
