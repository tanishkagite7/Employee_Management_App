# Employee Management App - Frontend

This is the **ReactJS frontend** for the Employee Management Application. It provides a user interface to manage employees and interacts with the Spring Boot backend via REST APIs.

---

## Features

- Add, view, edit, and delete employees
- Responsive UI using Bootstrap
- API calls handled with Axios
- Runs on **Node.js** and **npm**

---

## Tech Stack

- ReactJS
- Axios
- Bootstrap
- Node.js
- npm

---

## Prerequisites

- Node.js 16+
- npm 8+
- Backend API running on **http://localhost:8084**
