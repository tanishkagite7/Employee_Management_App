# Employee Management App

This is a **Full-Stack Employee Management Application** built using **Spring Boot** for the backend and **ReactJS** for the frontend. The app allows users to perform CRUD (Create, Read, Update, Delete) operations on employee records.

---

## Features

- Add new employees  
- View all employees  
- Update employee details  
- Delete employees  
- Responsive and interactive UI  

---

## Tech Stack

**Frontend:**  
- ReactJS  
- Axios (for API calls)  
- Bootstrap (for UI styling)  

**Backend:**  
- Java 17  
- Spring Boot  
- Spring Data JPA  
- MySQL (or any relational database)  
- Maven (for dependency management)  

---

## Prerequisites

- Java 17 or higher  
- Maven 3.9 or higher  
- Node.js and npm  
- MySQL or any relational database
